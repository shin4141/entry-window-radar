# Market Horizons

Use this file to describe source-backed market horizons that may matter for several projects.

Market Horizon is not prediction. It is a source-backed context layer for future Portfolio Entry Horizon comparison.

Important:

- Market size must be source-backed when used.
- Do not invent market size.
- If no source exists, write "unknown" and mark Source confidence low.

## Market Horizon: <name 1>

- Horizon year: The year or time range this market signal refers to.
- Market size estimate: Source-backed estimate only, or "unknown".
- Source: Citation, report name, URL, or internal source note. Use "none yet" if missing.
- Source confidence: High, medium, low, or unknown. Mark low when the source is weak or absent.
- Adjacent markets: Nearby categories, workflows, or budgets that may pull this project forward.
- Timing signal: Current evidence of urgency, buyer movement, category momentum, regulation, platform shift, or operational pain.
- Relevance to projects: Which project names this horizon may affect, and why.
- Notes: Caveats, missing evidence, or reasons this horizon should not be overread.
