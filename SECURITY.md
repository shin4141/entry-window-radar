# Security Policy

Do not open public issues containing secrets, credentials, private repo content, local paths, personal information, or unpublished strategy.

This repo does not currently run external APIs, runtime automation, hooks, MCP, pluginization, PDF generation, screenshot automation, scoring, or snapshot comparison.

Security reports should describe the issue without exposing sensitive material.

Runtime/security expectations should not be inferred from prompt examples, visual prototypes, or future-spec documents.

Open a private channel with the maintainer if available; otherwise avoid posting sensitive details publicly.
