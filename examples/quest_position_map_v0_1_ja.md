# Quest Position Map v0.1 日本語例

## 目的

この例は `outputs/quest_position_map_ja.svg` の読み方を残すためのスナップショット補助メモです。

SVG はすべての概念を単体で説明するためのものではありません。

グラフは位置を示します。

選択中のクエストカードが、なぜその位置なのかを説明します。

このスナップショットは、未来の Codex / AI セッションが As-of の解釈を失わずに再開するための文脈です。

## Codex Interpretation

- This SVG is not intended to explain every concept alone.
- The graph shows position.
- The card explains why.
- The snapshot preserves the As-of interpretation for future Codex/AI sessions.
- Future Codex should read the selected card before modifying scope.

## 選択中クエスト

### A. AI低導入・反復業務圧縮

- 推奨: PROOF
- 位置: 左上寄り
- 意味: 証拠まで比較的近く、長期化の可能性もある
- 未証明: 成果物の同等性 / 有料リピート
- 次の一手: 小さな比較試用を1回行う
- 広げてはいけない境界: 有料リピートまたは同等証拠が出る前に汎用化しない

## 読み方

- x軸: 報酬までの距離。右に行くほど遠い。
- y軸: 長期チャンス。上に行くほど長く棲める / チャンスが大きい。
- 点: クエスト。
- リング: 敵 / 競争圧。
- 赤三角: 生存 / 関係リスク。
