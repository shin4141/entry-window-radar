# Support

Entry Window Radar is a research/prototype prompt-first repo. Support is not guaranteed.

For usage, start with:

- `README.md`
- `prompts/quest_snapshot_generator_prompt_v0_1.md`

Open an issue only if it does not include private project data, credentials, local paths, unpublished strategy, or sensitive repo content.

Do not request investment advice, market prediction, launch permission, automatic decision-making, runtime automation, external posting, or execution-engine behavior.
