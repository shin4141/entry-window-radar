# Code of Conduct

Entry Window Radar is a prompt-first, template-first project for preserving project state, Gate, next action, and boundaries across AI sessions.

Participation should be respectful, specific, and non-harassing.

Do not use issues, pull requests, or discussions for:

- harassment, personal attacks, or discriminatory language
- spam or promotional material
- misleading claims that this repo provides runtime automation, market prediction, launch permission, investment advice, or an execution engine
- attempts to bypass the current Gate or Do-Not-Do Boundary

The maintainer may close or moderate issues and pull requests that ignore project boundaries, include sensitive material, or broaden scope without an explicit Gate.
